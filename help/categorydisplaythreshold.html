Which Categories To Announce on Access Denied

This option only applies to pages blocked by weighted phrase filtering. It defines the minimum score that must be accumulated within a particular category in order for it to show up on the access denied pages' category list.

Regardless of the setting, all categories under which the page scores positively will be logged; those that were not displayed to the user will appear in brackets in the log.

Possible values are:

  • -1 : display only the highest scoring category on the access denied page
  • 0 : display all categories on the access denied page (default)
  • n : minimum score for a category to be displayed on the access denied page
