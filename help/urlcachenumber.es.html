Número de Scanned-Clean URL a Caché
  

Caches buenas páginas para que ellos no necesitan ser escaneada de nuevo. También trabaja con plugins AV.

  • 0 = desactivado (recomendado para proveedores de servicios de Internet con los usuarios con disimilar la navegación)
  • 1000 = recomendado para la mayoría de los usuarios
  • 5000 = límite máximo sugerido
  • Si está usando un plugin de AV a continuación, utilice, al menos, 5000.
