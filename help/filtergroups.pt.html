Número de Filtro Grupos
  

Esta é a pedra angular opção que gira em torno de todo o filtros múltiplos grupos mecanismo em DansGuardian. Defina-o para o número de grupos que você deseja filtrar. Cada um desses grupos filtro será então configurada por um dansguardianfN. conf file onde N é o filtro de grupo número (1-99).

Se esta opção for configurada para apenas 1, filtros múltiplos grupos de todo o mecanismo vai ser efectivamente desligada. Todos os pedidos usuário será sempre atribuído ao grupo filtro f1, e apenas dansguardianf1.conf será significativo.
