Idioma Directorio
  

Dir el idioma en que se almacenan los idiomas para la internacionalización. La plantilla HTML dentro de este directorio sólo se utiliza cuando reportinglevel se fija a 3. Cuando se utiliza, DansGuardian mostrará el archivo HTML en lugar de utilizando el script de perl cgi. Esta opción es más rápida, más limpia y más fácil de personalizar la página de acceso denegado. El archivo de idioma se utiliza sin importar lo que sin embargo el establecimiento. Ejemplo: / etc / dansguardian / idiomas
