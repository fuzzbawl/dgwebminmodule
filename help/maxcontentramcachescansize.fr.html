Max RAM contenu scan taille du cache
  

Ce n'est utilisée que si vous utilisez un scanner de contenu tels que le plugin AV C'est la taille maximale des fichiers que vous téléchargez et DG cache dans la RAM. Après cette limite est atteinte, il se cache sur le disque Cette valeur doit être inférieur ou égal à maxcontentfilecachescansize.

La taille est en kilo-octets - par exemple, 2048 = 2Mb
utilisation de 0 à régler les mêmes que maxcontentfilecachescansize.

Cette option mai être ignorés par le gestionnaire de téléchargement configuré.
