Alguns de funcionamento do DansGuardian é sub-contratada para "plugins". Plugins são pedaços de código que invoca DansGuardian para fornecer funcionalidade adicional e estão quase completamente separado a partir do DansGuardian base. Utilização de plugins dispõe:

  • fácil extensibilidade, mesmo, em teoria, permitindo que usuário escrito extensões
  • código fonte mais fácil manutenção
  • uma interface de usuário mais compartimentada

Atualmente plugins são utilizados para fornecer funcionalidade para:

  • authplugin (identificar os utilizadores ou computadores, para a atribuição de um filtro grupo)
  • downloadmanager (aumentar o arquivo baixar experiência)
  • contentscanner (adicional de filtragem, geralmente anti-vírus)
Cada uma destas sub-plugins tem uma configuração própria. Embora a maioria destes sub-configurações são mínimas e nunca precisam mudar, de vez em quando uma dessas sub-configurações não precisam ajustar. Utilize esta ferramenta quando um plugin sub-configuração necessidades modificação.
