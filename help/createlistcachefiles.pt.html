Criar lista cache arquivos
  

Após o processamento (tipicamente triagem) lista cada arquivo, criar um correspondente cache (. transformados) arquivo. Ao reiniciar, compare a data do carimbo do arquivo de cache carimbo com a data da lista original arquivo e recriar o arquivo de cache somente se a lista inicial foi alterada recentemente. Caso use o cache (. Transformados)

Isto irá acelerar consideravelmente DansGuardian início em computadores mais lentos. A três vezes (300%) melhora na inicialização velocidade é típico. Fast computadores, na verdade, não precisam desta opção, mas ele não machucou nada.

(Note que o cache arquivos corrompidos, que são extremamente raros, mas que ocorrem uma vez em um grande tempo, às vezes pode impedir inicialização. Se o seu DansGuardian repente não será iniciado embora nada mudou, tente remover todo o cache [. transformados] arquivos.)
