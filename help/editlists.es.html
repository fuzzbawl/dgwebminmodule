ver o editar la configuración para todo el sistema de listas
  
  

El funcionamiento de DansGuardian es ampliamente configurado a través de varias listas (sitios, URL, extensiones, etc.) Sólo hay una copia de todo el sistema de algunas de estas listas, mientras que muchos otros tienen una copia para cada filtro de grupo. Esta herramienta le permite seleccionar uno de los de todo el sistema de listas y luego editarlo.

Si añade una nueva lista, darle un nombre cuya parte izquierda es el nombre de es padre lista (la lista que. Incluir es todo). Por ejemplo, si añade un archivo especial para algunos 'bannedsitelist' objetos, nombre es algo así como 'bannedsitelistproxies'. (Observe el patrón de 'bannedsitelistproxies'.)

Nuevos archivos con el nombre de esta manera se dará el mismo archivo de los permisos de acceso como su padre la lista, y se buscará al mismo tiempo que sus padres lista.
