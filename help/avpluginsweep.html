AntiVirus Plugin Option: Sweep
Where the Virus scanner is installed. This is the command needed to run it.

Note: If you want to use multiple virus scanners, then this should be a comma-separated list of commands, **in the same order** as they are listed in the "Virus Scanner" keyword just above. For example: 
    Sweep = /usr/local/Sophos/bin/sophoswrapper, /usr/local/f-prot/f-protwrapper
