Log file format

This is the format in which the log file will be written.

  • 1 = DansGuardian native log file format (space delimited)
  • 2 = CSV(Comma Separated Values)-style format (otherwise DansGuardian native)
  • 3 = Squid native format (the native format, not whatever format Squid happens to be using at the moment.
  • 4 = Tab-delimited (otherwise DansGuardian native)
