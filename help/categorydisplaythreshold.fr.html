Quelles catégories d'annoncer sur Access Denied
  

Cette option s'applique uniquement aux pages bloquées par pondérée expression de filtrage. Il définit le minimum qui doit être accumulé à l'intérieur d'un catégorie afin qu'elle apparaisse sur les pages d'accès refusé "la liste de cette catégorie.

Indépendamment de l'établissement, toutes les catégories dans lesquelles la page de résultats positifs seront enregistrés, les qui ne sont pas affichés à l'utilisateur apparaît entre parenthèses dans le journal.

Les valeurs possibles sont:

  • -1: affichage de la plus haute catégorie de notation sur la page d'accès refusé
  • 0: afficher toutes les catégories sur la page d'accès est refusé (par défaut)
  • n: score minimum pour une catégorie à afficher sur la page d'accès refusé
