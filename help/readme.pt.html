Requisitos

Ao contrário de versões anteriores deste módulo Webmin ou a maioria dos outros módulos, este módulo utiliza JavaScript fortemente. De fato, você provavelmente não vai mesmo ser capaz de usar este módulo Webmin se você desligar o JavaScript no seu navegador.

Esta versão deste módulo centra-se em user-friendliness à custa de um aumento substancial das exigências da CPU e memória RAM. Desde este módulo é executado em um computador moderno com adequada CPU e memória RAM, tudo funcionará bem. Mas este módulo provavelmente não irá desempenhar adequadamente sobre um velho lento pequeno computador, mesmo que corriam versões anteriores deste módulo ok.

Secundário Acessórios

Tal como antes, normalmente config são editadas com menus que são apresentados separadamente cada opção. Agora, se alguma coisa correr mal com os menus, é sempre possível para editar todos os arquivos diretamente e configuração manualmente também. Não há cada vez mais um motivo para recorrer a um não-Webmin ferramenta, porque a configuração está "preso".

Salvar automático e capacidade para reverter a cada versão de cada ficheiro config ou lista. Sempre a possibilidade de voltar para uma versão anterior desta forma é uma espécie de "desfazer" característica.

Quase todas as condições de erro incluir não apenas um usuário notificação mas também indicações sobre a forma de resolver o problema. Quando muda para o Webmin "módulo config" são sugeridos, um hiperlink para o "módulo config" está incluído direito na mensagem de aviso. Os usuários não precisam adivinhar ou a clicar em outra coisa em outro lugar. Em um efeito universal "desfazer" funcionalidade já está disponível.

Traduções

Vários idiomas são suportados usando o Webmin mecanismo padrão. Embora pobre, é esperado, no entanto, estas traduções são bom o suficiente para ser útil. Se você não gostar de um transation, fazer uma melhor um e que contribuam para a inclusão em uma futura versão deste módulo. Você pode substituir apenas algumas partes de uma tradução já existente. Não há necessidade de substituir a coisa toda de uma vez.

Ao que parece, um malaio tradução seria especialmente útil, mas infelizmente não é ainda disponível. Se um parcial malaio tradução foram contribuíram, seria utilizada sempre que estava disponível, eo padrão do Webmin mecanismo seria automaticamente retroceder para outro idioma em outros lugares.

Unattended Lote Operação
Embora a produção de relatórios é interativo mais obviamente feito através do Webmin, reconhece este módulo, há uma necessidade de autônoma lote geração automática de relatórios também. Assim, todos os relatórios que o mesmo pode ser produzido interativamente também pode ser produzido em um lote ambiente. Como configurar a execução automática e distribuição de relatórios pode ser um pouco complicado, um script e instruções detalhadas estão incluídos. A clickable link logo abaixo do menu à mesma página que lança interactivo Análise Log pode ser usado de escrever uma cópia do relatório a amostra do lote script sempre que você desejar.

filtros múltiplos Grupos

Criação de filtros múltiplos grupos pode ser intimidante, como a forma mais simples de fazê-lo cria uma enorme quantidade de arquivos todos em um diretório. Portanto, este módulo inclui uma alternativa automatizada. Basta especificar o que deseja que o filtro de grupos para ser nomeado e como você deseja que os arquivos a serem relacionados, e esta ferramenta irá criar ou copiar todos os arquivos necessários. Além disso, um bloco comentário explicando exactamente o que cada arquivo é para serão adicionados automaticamente ao seu topo.

Esta configuração só precisa ser feito uma vez, após o qual a opção de configuração deste módulo será apresentada em cinza. Mas, fazendo isso mesmo só uma vez é tão importante e pode ser tão onerosa que este módulo pode fazer isso para você automaticamente.

Pesquisa detalhada Acessórios

A pesquisa está agora a sua própria ferramenta em vez de ser uma função enxertado qualquer outro instrumento.

Uma grande variedade de coisas que podem ser pesquisados para inteligentemente: palavras, frases, ao domínio (host) nome, URL, e mesmo padrão. Pode-se procurar e encontrar qualquer expressão regular apenas digitando a expressão regular literalmente. Não há necessidade de utilizar qualquer estranha sintaxe para pesquisar, Nem só de traduzir alguns caracteres. Procurando uma frase automaticamente toma conta possivelmente diferentes palavra de ordem. Se você digitar apenas "Jack saltou" irá automaticamente pesquisa por "saltou Jack" também.

Para cada arquivo no qual um ou mais jogos foi encontrado, ou não o arquivo filtro aplica-se a cada grupo serão apresentados. Portanto, se o conteúdo de um arquivo se aplicam a alguns grupos filtro mas não para outros, ele será imediatamente claro. Filtrar grupos estão listados pelo nome do grupo filtro (se o grupo tem um nome), tornando mais fácil para compreender o visor apenas por olhos nisso.

Terciário Acessórios

Page cabeçalhos e rodapés são mais consistentes, como é a exibição de mensagens de alerta. Tanto a versão e os DansGuardian Webmin Módulo versão são exibidos em cada página, e sempre no mesmo lugar.

Testando o ambiente e visualização de mensagens de alerta quando algo não parece certo foram grandemente ampliadas.

Cada linha em cada arquivo modificado é comentado com uma data-e-hora carimbo indicando quando foi alterado.

Inserção das opções existentes no config é feita semi-inteligente, tão novas opções não tudo amontoam no final do arquivo. Se a opção existia anteriormente mas foi, então, comentou a e agora está a ser adicionado de volta, a nova opção será imediatamente o antigo comentário em qualquer parte do arquivo.

Sempre filtros múltiplos grupos existentes, todos os grupos estão listados filtro lado a lado (muitas vezes como guias na mesma página). E eles são, maioritariamente referido pelo seu nome atribuído em vez de 'f1', 'f2', 'f3', e assim por diante.

Para modificar o arquivo de configuração, opções de configuração são repartidos por um grande número de menu separadores. Para aqueles momentos em que você sabe o nome da opção pretende modificar mas não sei o que ele está no guia, há uma opção localizador, que enumera todas as opções e salta imediatamente para a direita quando você seleciona uma guia.

Cada ambiente limitação, como a não DansGuardian executável ou não DansGuardian config arquivo, desactiva apenas os instrumentos relevantes. Limitações deixaram desactivar todo o módulo assim você não pode executar qualquer coisa.

Todas as opções mostradas plugin mesmo se actualmente comentou fora.

Alguns padrões que variam de um sistema para o próximo porque são elaboradas, no entanto, estão em correctamente reflectidos na configuração modificação ferramentas. Em particular muitos caminhos sistema de arquivos padrão são personalizados de acordo com o local DansGuardian. Se um pacote diferente DansGuardian construída com configurações diferentes é obtida, Este módulo irá tentar ajustar a refletir isso.

Tentativa de ajustar automaticamente a diferentes caminhos (tais como /usr/local/etc/dansguardian/... x /etc/dansguardian/...). Fornecer a funcionalidade completa, independentemente do que caminhos que estão em uso.

Avance para o ecrã inicial mesmo caminhos estão incorretas. A partir daí, o apoio do Webmin usando para editar o "módulo config" como necessário especificar os caminhos corretos.

Opção valores são cotados em config inteligência. Números e não curto espaço cordas não estão fechados entre aspas, enquanto mais cordas e cordas com espaços são.

Como antes, todos os ficheiros irá ser lido e entendido independentemente de eles usam DOS line endings ou Unix linha terminações. O que é novo é que todos os arquivos modificados serão sempre por escrito com o Unix linha terminações, mesmo que tenham sido modificadas dentro de um navegador web rodando em um computador Windows.
