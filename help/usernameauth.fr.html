méthodes d'identification d'utilisateur
  
Vous pouvez avoir autant de méthodes que vous voulez, et non pas un seul. Le premier sera utilisé, et si aucun nom d'utilisateur a été trouvé, la prochaine sera utilisé.

Proxy Auth
Proxyauth est alors de base pour l'authentification de proxy est utilisé (pas bon pour les proxy transparent).

Ident Auth
Ident est pour quand les autres ne fonctionnent pas. Il prendra contact avec l'ordinateur qui venait de la connexion et de requête pour le propriétaire de la connexion. Un serveur ident / démon est nécessaire pour être en cours d'exécution sur l'ordinateur.
S'il vous plaît voir Identd pour Win9x pour un démon identd pour la plate-forme Windows.
