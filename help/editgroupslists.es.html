ver o editar la configuración del filtro de Grupo Listas
  
  

La operación de DansGuardian es ampliamente configurado a través de varias listas (sitios, URL, extensiones, etc.) Sólo hay una copia de todo el sistema de algunas de estas listas, mientras que muchos otros tienen una copia para cada filtro de grupo. Esta herramienta le permite seleccionar uno de los grupos existentes de filtro, a continuación, seleccione una de sus listas, y luego editarlo.

La mayor parte de la rutina de configuración de DansGuardian se hará uso de esta herramienta.

La aparición de estas listas serán radicalmente diferentes dependiendo de la configuración de la "Incluir" fijos "en las listas de muestra" opción de configuración. Haga clic en "módulo de configuración" en la parte superior izquierda, y la tapa el establecimiento de otra manera para que puedas ver tanto las apariencias. A continuación, seleccione el ajuste que usted prefiera y configurar el módulo de configuración de la opción adecuada.

Si añade una nueva lista, darle un nombre cuya parte izquierda es el nombre de es padre lista (la lista que. Incluir es todo). Por ejemplo, si añade un archivo especial para algunos 'bannedsitelist' objetos, nombre es algo así como 'bannedsitelistproxies'. (Observe el patrón de 'bannedsitelistproxies'.)

Nuevos archivos con el nombre de esta manera se dará el mismo archivo de los permisos de acceso como su padre la lista, y se buscará al mismo tiempo que sus padres lista.
