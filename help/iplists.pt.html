Banidos / IP Listas Exceção
  

Esses arquivos fornecem um rápido e de forma suja para obter DansGuardian especial para lidar com computadores uma maneira particular, geralmente por períodos curtos de tempo. Se você encontrar um computador foi infectado pelo malware, esses arquivos oferecem uma maneira para instantaneamente desligar o computador de acesso à Web até que possa ser limpo.

(Estes arquivos não são um substituto para uma boa configuração de 'auth' métodos.)
