temporária negado / Infected Page Bypass
  

Isto fornece um link na página negado para contornar a proibição de alguns minutos. Ser seguro que usa um aleatório hashed secreta gerada pelo demónio inicialização. Você define o número de segundos para o bypass vai funcionar antes de o negar irá aparecer novamente.

Para permitir que o link da página negou a aparecer, você precisará mudar esta opção, não só mas também editar a template.html.

Os valores possíveis são:

  • 0: desabilitar a opção ignorar totalmente
  • -1: permitir que a derivação opção, mas exigem a separação dansguardian.pl-como programa CGI para gerar um link válido (para fazer bypass chave separadas geração possível, o bypasskey opção não deve ser um valor aleatório)
  • n: segundos para que o bypass opção permanecerá ativado (exemplo: 300 é de 5 minutos)

    A mesma funcionalidade está prevista para as páginas que vírus são infectadas (em vez de negada por causa do conteúdo) pela configuração opção infectedbypass, com as mesmas configurações possíveis.
