Set-Up Config-&-List Files Para Múltiplas Filtrar Grupos
  

Este processo deverá ser executado uma vez quando filtros múltiplos grupos são executadas primeiro. Depois filtros múltiplos grupos são criados, este processo nunca vai ter de ser realizado novamente.

Criação de todos os arquivos de filtros múltiplos grupos sem este auxílio pode ser bastante assustador. Isso torna o processo de configuração mais simples.

Mesmo que é automatizada, o processo é um pouco desestabilizante. Por isso, não deve ser realizada DansGuardian quando estiver executando nem quando você precisará usar DansGuardian para a produção em apenas alguns minutos.

Você será solicitado para vários bits de informação assim o processo sabe como configurar filtros múltiplos grupos para o seu ambiente. Você será solicitado sheme geral que pretende utilizar, se pretende anular um filtro grupo para "não" acesso à Web, se pretende anular um filtro para o grupo "todos" acesso à Web, filtrar o que cada grupo deve ser o nome, e se várias listas deverão ser armazenados por filtro grupo ou todos os grupos devem usar filtro uma lista compartilhada.

Os possíveis regimes gerais são:

  • 1: lista aninhada arquivos

    isto significa cada lista arquivo afectarão não só o seu próprio filtro grupo, mas também todos os outros grupos filtro abaixo (ou acima)-lo. Assim, por exemplo, se o seu filtro grupos são "tempo", "empregados", "gestores", e "executivos", qualquer mudança para a excepção ... listas de "gestores" irá também automaticamente afecta "executivos", e quaisquer alterações à proibidos ... listas de "gestores" também serão automaticamente afectar "empregados" e "tempo".

  • 2: lista separada arquivos

    isto significa cada filtro grupo, será dada próprias listas completamente independente, que não afecta nem ser afectadas por qualquer outro filtro grupo

  • 3: lista comum arquivos

    isto significa cada filtro grupo tem as suas próprias listas, ainda que também referências, que é uma lista comum filtro para todos os grupos. Em outras palavras, são comuns "base", lista na qual cada filtro grupos então elaboradas.
