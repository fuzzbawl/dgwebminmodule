Ponderação Embutido URL
  

Quando definido para algo maior do que zero, esta opção faz URLs embutidos dentro de um HTML da página (a partir de links, imagens tags, etc) para ser extraído e confrontados com os bannedsitelist e bannedurllist. Cada um link para uma página causas proibiu o montante fixado aqui para ser adicionado à página da ponderação. O comportamento desta opção no que se refere às múltiplas ocorrências de um site / URL é afectados pela weightedphrasemode definição.

  • 0: desativar opção
  • n: pontuação para cada proibido hiperlink

(Atualmente, esse recurso usa expressões regulares que necessitam da biblioteca PCRE. Como tal, ela só está disponível se o seu DansGuardian foi construída para incluir PCRE apoio. Você pode verificar executando dansguardian-v e procurando quer uma definição para - enable-pcre =yes. ou a ausência de qualquer - enable-pcre instalação.)

(Esta opção é altamente intensivo da CPU.)
