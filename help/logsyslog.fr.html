Ecrire non-Access Log entrées dans Syslog
  

Ecrire non-accès dans les messages syslog. (Ce réglage n'affecte pas l'accès des messages, qui sera écrit dans le fichier journal, indépendamment dansguardian.)

La destination réelle de ces messages est déterminé par votre configuration Syslog. Les messages produits par dansguardian ont une facilité d'UTILISATEUR. La plupart des messages produits par un dansguardian niveau de ERR. Certains ont un niveau d'INFO. Bien que d'autres niveaux ne sont pas actuellement utilisés, ils sont possibles dans l'avenir.
