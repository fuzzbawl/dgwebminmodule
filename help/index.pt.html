DansGuardian
  

DansGuardian é um filtro conteúdo da Web que atualmente funciona em Distribuições Linux, FreeBSD / OpenBSD, e Solaris. DansGuardian não apenas filtro puramente baseado em uma lista de sites proibidos como muitos menor (e totalmente comercial) filtros. Isso vai além da habilitação para URL filtrar o conteúdo real de páginas web usando muitos métodos, como a filtragem PICS e, especialmente, sublinhando a sua frase ponderada correspondente. Ponderado frase correspondência pontuação conteúdo da página um conjunto muito mais inteligente do que simplesmente "sim má palavra visto" ou "não vi nenhuma palavra", melhorando, assim, grandemente filtragem decisões.

DansGuardian é projetado para ser completamente flexível e permite que você personalizar a filtragem para a sua exata necessidades. Pode ser tão draconiano ou como unobstructive que quiser. Predefinições —orientada para que uma escola primária pode querer— são fornecidos para você começar. Uma vez que o seu DansGuardian é totalmente instalado, você está no controle do que você deseja bloquear. Você pode usar este Webmin módulo para ajustar facilmente DansGuardian a configuração para os seus desejos.

DansGuardian é um verdadeiro conteúdo web filtro.

(loosely extraído de uma mais detalhada introdução no DansGuardian Site.)

Este Webmin módulo fornece gráfica gestão do 2,10(2,9) série de DansGuardian lançamentos. Ela enfatiza a Multilple Filtro Grupos característica do DansGuardian e detalhada Log Análise filtragem e relatórios. O Log Análise pode ser executado em um diferentes computador e/ou pode produzir relatórios em um lote autônomo Modo (um exemplo arquivo script é fornecido), bem como sobre o funcionamento interativamente DansGuardian servidor. Este Webmin módulo também fornece arquivos exemplo para a execução de uma Master→Slaves configuração onde apenas um site é administrado enquanto as outras são simplesmente "clones".

Ao contrário de muitos outros Webmin módulos, este DansGuardian Webmin módulo utiliza um lote de JavaScript. Na verdade, ele pode até não ser utilizado se o JavaScript está desativado. Além disso, este módulo requer significativamente mais CPU e memória RAM do que as versões anteriores. Seja cauteloso sobre a publicá-lo no mesmo computador que o DansGuardian durante a produção horas.
