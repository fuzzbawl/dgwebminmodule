Web acesso negado relatórios
  

As opções definidas aqui para Web acesso negado relatórios não afectam madeireira.
Reporting level:

  • -1 = log, mas não bloqueiam (Stealth mode)
  • 0 = Basta dizer "Acesso Negado"
  • 1 = Relatório porquê, mas não negou que frase
  • 2 = Relatório plenamente
  • 3 = Utilizar Modelo arquivo HTML (Acesso Negado Endereço ignorado)
