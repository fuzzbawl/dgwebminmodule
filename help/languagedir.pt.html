Language Directory
  

Língua dir onde as línguas são armazenados para internacionalização. O HTML modelo dentro desta dir só é utilizado quando reportinglevel é definido como 3. Quando utilizado, DansGuardian irá exibir o arquivo HTML em vez de utilizando o perl script cgi. Esta opção é mais rápido, mais limpo e mais fácil de personalizar o acesso negado página. A língua é usada arquivo não importa o cenário no entanto. Exemplo: / etc / dansguardian / languages
