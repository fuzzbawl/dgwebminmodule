Langue Répertoire
  

Langue dir où sont stockées les langues de l'internationalisation. Le code HTML de ce modèle dir est utilisé seulement lorsque reportinglevel est fixé à 3. Lorsqu'il est utilisé, dansguardian affichera le fichier HTML au lieu de en utilisant le script perl cgi. Cette option est plus rapide, plus propre et plus facile à personnaliser la page d'accès refusé. Le fichier de langue est utilisée, quelle que soit cependant la mise. Exemple: / etc / dansguardian / langues
