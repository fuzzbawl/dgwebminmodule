Esta opção só se aplica a páginas bloqueadas pelo ponderada frase filtragem. Ele define a pontuação mínima que deve ser acumuladas dentro de um determinado categoria para que possa mostrar-se sobre o acesso negado páginas "categoria lista.

Independentemente da configuração, todas as categorias em que a página pontuação positiva será autenticado; essas que não foram exibidas para o usuário irá aparecer entre parênteses no log.

Os valores possíveis são:

  • -1: exibir apenas a pontuação mais alta categoria na página de acesso negado
  • 0: mostrar todas as categorias de acesso negado página (padrão)
  • n: uma pontuação mínima para a categoria a ser exibido na página acesso negado
