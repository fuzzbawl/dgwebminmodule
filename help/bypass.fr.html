Refus temporaire / Infected Page Bypass
  

Cela donne un lien sur la page refusé de contourner l'interdiction de quelques minutes. Être sûr, il utilise un hasard haché secret généré à démon démarrage. Vous définissez les nombre de secondes de la dérivation de fonctionnement de nier devant les apparaîtra de nouveau.

Pour permettre le lien sur la page refusé de comparaître, vous devrez non seulement de modifier cette option mais aussi modifier le template.html.

Les valeurs possibles sont:

  • 0: désactiver l'option bypass entièrement
  • -1: permettre à l'option de contournement, mais il faut le séparer dansguardian.pl comme programme CGI pour générer un lien valide (pour faire court-circuiter la génération de clés distinctes possible, la bypasskey option doit être une non-valeur aléatoire)
  • n: pour la seconde option qui la rocade restera activé (exemple: 300 est de 5 minutes)

    La même fonctionnalité est fournie pour les pages qui sont infectés par le virus (plutôt que de refuser en raison du contenu) par l'option de configuration infectedbypass, avec les mêmes réglages possibles.
