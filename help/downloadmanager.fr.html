gestionnaire de téléchargement de plug-ins
  

Ce gestionnaire de téléchargement des plugins fournir à l'utilisateur différents types d'amélioration de l'expérience lors du téléchargement de fichiers sur Internet. Normalement, tout ce qui est nécessaire pour activer / désactiver (décommentez / commentaire) individuelles, qui peut être fait par le "plugins" dans l'onglet de configuration de montage principale tâche fenêtre.

Chaque plugin a une configuration qui lui est propre. Dans la plupart des cas, ces configurations n'ont pas besoin d'être changé, Si l'une des configs ne doivent être modifiées, vous pouvez accéder au fichier de configuration pour l'édition par le biais du plugin config outil d'édition.

Certaines des options mai dire #!! Non compilé! . Si tel est le cas, aucune combinaison de configuration ne peut jamais permettre à l'une de ces options sur votre système. Afin de rendre ces options disponibles, vous devez obtenir un exécutable dansguardian qui inclut le support de l'option et les fichiers de configuration que l'égaler. Une façon de le faire est de reconstruire dansguardian de source. La reconstruction inclut l'exécution. / Configure, la compilation, de liens, et l'installation; il ne doit pas être pris à la légère.
