Reverse lookups interdit de site et les URL
  
Si activée, dansguardian sera en place avant le DNS pour un IP adresse URL et de recherche à la fois pour les interdits dans le site Web et de listes. Cela permettrait d'éviter un utilisateur de simplement entrer dans la propriété intellectuelle pour une adresse interdite. Il permettra de réduire quelque peu la recherche de vitesse, sauf si vous avez un serveur DNS cache local, laissez-le et utilisez l'option de couverture Bloc IP dans le fichier au lieu bannedsitelist.
