Filtro de contenido máximo tamaño de página A veces, los servidores web etiqueta archivos binarios como texto que puede ser muy lo que provoca una gran pérdida de enormes recursos de CPU y memoria. Para contrarrestar esto, se puede limitar el tamaño del documento que se filtrada y conseguir que acaba de pasar directamente a través de. Este ajuste se aplica también al contenido de expresiones regulares modificación.

El tamaño es en Kilobytes - 2048 por ejemplo, = 2Mb
