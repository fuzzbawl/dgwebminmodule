Max content RAM scan cache size

This is only used if you use a content scanner plugin such as AV This is the max size of file that DG will download and cache in RAM. After this limit is reached it will cache to disk This value must be less than or equal to maxcontentfilecachescansize.

The size is in Kilobytes - eg 2048 = 2Mb
use 0 to set it the same as maxcontentfilecachescansize.

This option may be ignored by the configured download manager.
