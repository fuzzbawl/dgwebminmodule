Frase modo ponderado
  

Existen 3 posibles modos de funcionamiento:

  • 0 (apagado) = no utilizar la frase ponderado en función de todos.
  • 1 (en normal) = normal ponderado frase operación, contar cada frase ponderado con la frecuencia que ocurre en una página web.
  • 2 (en singular) = suplente peso puntuación método de cálculo, contar cada frase ponderada sólo una vez [la primera vez] que aparece en una página web.

En cuanto la característica ponderada frase completamente fuera de lobotomize DansGuardian de "verdadera" (o "contenido") de filtrado web a poco más de lujo (incluyendo expresiones regulares) Complementado por el filtrado de URL prohibidas las frases y la posibilidad de modificar el contenido, que no es muy diferente de los filtros de tela mucho más sencillo.

DansGuardian soporta dos métodos diferentes de cálculo de resultados de peso, "normal" y el método de "singular" método. Cada método tiene sus pros y sus contras. Páginas web de muy buena puntuación y con cualquier método, y muy mala puntuación a las páginas web mal con cualquiera de los métodos, pero la puntuación media de páginas web mayo debajo de su umbral con un método, pero por encima de su límite con el otro. Así que no se estableció un cambio de sistema DansGuardian para usar el otro método la ligera.

Además, los dos métodos de calificación de peso puede requerir algo diferente "naughtyness" límites con el fin de comportarse de manera similar. Si el valor que hayas elegido para "naughtynesslimit" es sólo una idea aproximada adivinar, no importa. Pero si ha sintonizado "naughtynesslimit" dentro de un margen de sólo 5 o 10, es posible que tenga que volver a ajustarlo después de cambiar el método de calificación de peso.
