Max conteúdo arquivo digitalizar tamanho do cache
  

Isto só é usado se você usar um plugin como o conteúdo scanner AV. Este é o tamanho máximo arquivo que irá baixar DG para que possa ser digitalizada ou vírus controlado. Este valor deve ser maior ou igual a maxcontentramcachescansize.

O tamanho é em Quilobytes - por exemplo, 2048 = 2Mb
