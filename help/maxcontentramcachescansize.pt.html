Max conteúdo RAM scan tamanho do cache
  

Isto só é usado se você usar um plugin como o conteúdo scanner AV Este é o tamanho máximo do arquivo que irá baixar e DG cache na RAM. Após este limite for atingido vai cache de disco Este valor deve ser inferior ou igual a maxcontentfilecachescansize.

O tamanho é em Quilobytes - por exemplo, 2048 = 2Mb
utilização de 0 a defini-la da mesma forma que maxcontentfilecachescansize.

Esta opção pode ser ignorada por configurado o gerente do download.
