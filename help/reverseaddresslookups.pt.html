Reverse lookups para proibidas site e URLs
  
Se estiver definido para ontário, DansGuardian vai olhar para cima a frente DNS para um endereço IP e pesquise tanto no banido do site eo URL listas. Isto impediria que um usuário de simplesmente digitando o endereço IP para um proibida. Irá reduzir velocidade pesquisar algo assim se você tiver um cache local do servidor DNS, deixe-o desligado e use a opção Block Blanket IP no arquivo bannedsitelist vez.
