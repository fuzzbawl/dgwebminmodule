Daemon Usuário / Grupo
  
Como todos os processos, o DansGuardian servidor / daemon processo tem um usuário e grupo. O usuário / grupo de cada processo controla a forma como o sistema operacional concede permissões ao mesmo, nomeadamente o acesso a vários arquivos e diretórios. Tipicamente operação é mais simples DansGuardian caso, o seu representante local backend (por exemplo, Lula), e de qualquer anti-vírus daemon tudo correr como o mesmo usuário / grupo

Se não for especificado, será padronizado DansGuardian para correr como qualquer usuário e grupo foram construídas em (normalmente ninguém quer / ninguém ou nouser / nogroup). É provavelmente melhor para especificar explititly alguns usuário userid / groupid do que a chance como arquivo permissões necessárias para fazer o trabalho construído em configurações.
