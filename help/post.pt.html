POST Protection (upload e formulários da Web)
  

Não se faz sem bloquear qualquer arquivo upload, ou seja, este é apenas para bloquear ou limitar os uploads medidos em kilobytes após cabeçalho MIME codificação e lombada

Utilize 0 para um bloco completo
Utilizar tamanho em kb (por exemplo, 512 = 512 Kilobytes) para limitar
Use -1 para Não bloqueio.
