visualizar ou editar System-Wide Configuração Listas
  
  

O funcionamento do DansGuardian é extensamente configurado através de várias listas (sites, URLs, ampliações, etc.) Existe apenas um sistema de escala cópia de algumas dessas listas, enquanto muitos outros têm um texto diferente para cada filtro grupo. Esta ferramenta permite que você selecione um sistema de escala listas e depois editá-lo.

Se você adicionar uma nova lista, dar-lhe um nome cujo lado esquerdo é o nome da é mãe lista (a lista. Incluir's it). Por exemplo, se você adicionar um arquivo especial para alguns 'bannedsitelist' itens, nome é algo como 'bannedsitelistproxies'. (Aviso do padrão 'bannedsitelistproxies').

Novos arquivos chamado desta forma, será dada o mesmo arquivo permissões de acesso como a sua mãe lista, e vai ser pesquisado, ao mesmo tempo que sua mãe lista. Se você criar uma nova lista Nome do arquivo, mas é outra forma, pesquisas não podem encontrar itens no novo arquivo, Webmin permissões e não pode permitir que você edite o arquivo, efetivamente tornando inútil,
