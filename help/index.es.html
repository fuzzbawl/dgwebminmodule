DansGuardian
  

DansGuardian es un filtro de contenido Web que se ejecuta en la actualidad Distribuciones de Linux, FreeBSD / OpenBSD y Solaris. DansGuardian filtro no sólo meramente sobre la base de una lista de sitios prohibidos al igual que muchos menos (y totalmente comercial) filtros. Va más allá de la simple investigación de la URL filtrar el contenido de las páginas web utilizando muchos métodos, tales como el filtrado FOTOS y sobre todo, centrándose en su ponderada frase correspondiente. Ponderada frase se pongan en venta un contenido de la página de resultados mucho más inteligente que sólo "Si la palabra mal visto" o "no mala palabra no visto", Así pues, en gran medida la mejora de las decisiones de filtrado.

DansGuardian está diseñado para ser completamente flexible y le permite adaptar el filtro a la medida de sus necesidades. Puede ser tan draconianas como unobstructive o como quieras. Ajustes por defecto —orientada hacia lo que uno puede querer a la escuela primaria— se proporcionan para empezar. Una vez que su DansGuardian esté completamente instalado, usted está en control de lo que desea bloquear. Puede utilizar esta Webmin módulo para ajustar fácilmente DansGuardian la configuración a sus deseos.

DansGuardian es un verdadero filtro de contenido web.

(Libremente extraídos de una introducción más detallada de la DansGuardian Sitio Web.)

Este Webmin módulo de gestión gráfica de los 2,10(2,9) serie de DansGuardian emisiones. Se hace hincapié en el Multilple Grupos Filtro de la función DansGuardian y detallada Registrarse Análisis de filtrado y la presentación de informes. El Registro Análisis se pueden ejecutar en un diferentes ordenador o/y puede producir los informes en un lote sin vigilancia el modo de (ejemplo un archivo de comandos se suministra), así como ejecutando DansGuardian interactivamente en el servidor. Este Webmin módulo también proporciona los archivos de ejemplo para la aplicación de una Master→esclavos configuración de un sitio donde sólo se administra como los otros son simplemente "clones".

A diferencia de muchos otros Webmin módulos, este DansGuardian Webmin módulo utiliza una gran cantidad de JavaScript. De hecho, es posible que no pueda utilizarse incluso si javascript está deshabilitado. Además, este módulo requiere mucho más CPU y memoria RAM que las versiones anteriores. Ser prudentes a la hora de ejecutarlo en el mismo equipo que DansGuardian durante horas de producción.
