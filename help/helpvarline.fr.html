Activer / Désactiver (Décommentez / Comment) Ligne A
  

Dans certains cas, toute la ligne doit être activée ou désactivée sans changer leur contenu. Ces options sont le plus souvent présenté avec une petite boîte. Une vérification dans la petite case signifie la ligne doit être activé (décommenté). Si la petite boîte est vide, cela signifie la ligne doit être désactivé (en commentaires).
