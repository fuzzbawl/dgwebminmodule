Download Manager plug-ins
  

Estes gerente do download plugins fornecem o usuário com vários tipos de maior experiência quando transferir ficheiros a partir da Internet. Normalmente tudo o que é necessário para activar / desactivar (descomentar / comentário) individuais queridos, que pode ser feito através do "plugins" guia nos principais config edição tarefa janela.

Cada plugin tem uma configuração própria. Na maioria dos casos, essas configurações não precisam ser alterados, Se uma das configs não necessitam de ser alterados, você pode acessar o arquivo de configuração para a edição através da ferramenta edição plugin config.

Algumas das opções podem dizer #! Não compilado! . Se assim for, não podemos jamais permitir que config combinação qualquer destas opções em seu sistema. Para fazer essas opções disponíveis, você deve obter tanto um executável DansGuardian que inclui suporte para a opção e config que igualar. Uma maneira de fazer isso é reconstruir DansGuardian da fonte. Reconstruindo inclui executando. / Configure, compilação, ligação e instalação; não deve ser empreendida de ânimo leve.
