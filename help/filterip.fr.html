Filtre de l'adresse IP
Les adresses IP écoutées par DansGuardian. Si le champ est vide, DansGuardian écoutera toutes les adresses IP de la machine. Ceci inclut modem, loopback, cartes réseau, etc. Normalement, un firewall protège ces entrées, mais vous pouvez restreindre l'écoute à une seule adresse IP. Oui, juste une.
