visualizar ou editar System-Wide Configuração
  
  

Ver ou editar as definições da maioria do sistema à escala DansGuardian opções de configuração. Estas são as definições que são (ou poderiam ser) no ficheiro config dansguardian.conf. Se você quiser ver ou editar as configurações de filtro individual grupos (as opções que são [ou poderia ser] no dansguardianfN.conf config) usar o "Ver/Editar um filtro do Grupo Base Config" ferramenta. Se você quiser ver ou editar uma configuração lista, usar o "Ver/Editar um filtro do Grupo Listas" ferramenta (ou, em alguns caso, o "Ver/Editar System-Wide Listas" ferramenta).

O sistema disponíveis à escala opções de configuração são agrupados em diferentes "separadores" nesta interface de utilizador. Desta forma, você pode manipular um grupo de opções relacionadas com enquanto não mudar (ou mesmo a olhar para) outras opções. (Se você souber o nome da opção que pretende manipular, mas não sei o que ele está no guia, utilize o separador finder no topo para ir para a direita guia automaticamente.)

A definição actualmente activa é mostrado com uma borda em torno dela. Como você faz as alterações propostas, a fronteira pode mover.

Para cada opção, o "Conjunto"? checkbox mostra ou não alguma definição para a opção está presente no arquivo de configuração. (Isso permite que os casos de valores vazios como 0 ou "off" e da ausência completa de qualquer especificação para a opção devem ser distinguidas claramente.) Para mudar ou não uma opção configuração está presente, clique sobre a caixa de seleção.

Se uma opção configuração é (ou será) presente, o "Sistema-Wide Valor" área irá mostrar o seu valor. O "Sistema-Wide Valor" área perto do lado esquerdo terá uma borda em torno dele para que você possa mais facilmente discernir quais opções são (ou serão) presentes. Se, por outro lado a opção Definir? é não marcada, Será feita referência a um valor padrão, se houver um. Na maioria dos casos, a fronteira vai passar para o "padrão" na área do direito. Você não pode alterar o padrão, mas você pode ler o que eles estão.

(O "default" é o valor-compilados em valor que serão realmente utilizadas, se a opção não é especificado. É muitas vezes não o valor recomendado nem a um defeito que aparece nos ficheiros de configuração.)

Sobre a "comunicação" guia, o 'htmltemplate' opção só será utilizada se o 'reportinglevel' opção está definida para "usar HTML arquivo modelo". Da mesma forma, o 'accessdeniedaddress' e 'nonstandarddelimiter' opção só será utilizada se o 'reportinglevel' opção é não definido como "uso HTML arquivo modelo". Filtro individual grupos podem sobrepor todas as opções sobre a "comunicação" guia, tão opções podem efectivamente ser utilizados noutras instalações embora esta exibição sugere que sejam utilizadas.
