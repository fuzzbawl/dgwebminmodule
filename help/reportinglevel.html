Web access denied reporting

The options set here for Web access denied reporting do not affect logging.
Reporting level:

  • -1 = log, but do not block (Stealth mode)
  • 0 = Just say 'Access Denied'
  • 1 = Report why but not what denied phrase
  • 2 = Report fully
  • 3 = Use HTML Template file (Access Denied Address ignored)
