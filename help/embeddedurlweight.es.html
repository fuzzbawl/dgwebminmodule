Ponderación Embedded URL
  

Cuando se establece en algo mayor que cero, esta opción hace que dentro de un URL incrustado HTML de la página (a partir de enlaces, imagen de las etiquetas, etc) que se extraen y se cotejarán con los bannedsitelist y bannedurllist. Cada enlace a una página causas prohibida la cuantía fijada aquí para ser añadido a la página de la ponderación. El comportamiento de esta opción con respecto a varios casos de un sitio / URL weightedphrasemode afectados por el ajuste.

  • 0: desactivar la opción
  • n: la puntuación para cada hipervínculo prohibido

(En la actualidad, esta función utiliza expresiones regulares que requieren de la biblioteca PCRE. Como tal, sólo está disponible si su DansGuardian fue construido a fin de incluir PCRE apoyo. Usted puede comprobar ejecutando dansguardian-v y buscando, ya sea un escenario para - enable-pcre =yes. o la ausencia de cualquier - enable-pcre de configuración.)

(Esta opción es muy intensivos en la CPU.)
