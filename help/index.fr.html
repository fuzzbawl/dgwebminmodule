dansguardian
  

Dansguardian est un filtre de contenu web, qui tourne actuellement sur Distributions Linux, FreeBSD / OpenBSD et Solaris. Dansguardian ne filtre pas purement basée sur une liste de sites interdits comme beaucoup moindre (et totalement commercial) des filtres. Il va au-delà du simple contrôle d'URL filtrer le contenu des pages Web en utilisant de nombreuses méthodes telles que le filtrage PICS et surtout en mettant l'accent sur ses pondérée expression matching. Pondéré phrase scores contenu de la page correspondant à un ensemble beaucoup plus intelligente que la simple "oui mauvais mot vu" ou "pas de mauvaise parole ne voit pas», ainsi grandement améliorer le filtrage des décisions.

Dansguardian est conçu pour être flexible et vous permet de adapter le filtrage à vos besoins. Il peut être aussi draconiennes que unobstructive ou que vous le souhaitez. Les paramètres par défaut —orienté vers une école primaire ce qui pourrait vouloir— sont fournis pour vous aider à démarrer. Une fois votre dansguardian est pleinement installé, vous en avez le contrôle de ce que vous voulez bloquer. Vous pouvez utiliser cette Webmin module d'adapter facilement dansguardian la configuration de vos envies.

Cette Webmin module de gestion graphique de l'2,10(2,9) dansguardian série de communiqués. Il met l'accent sur les Multilple Filter Groupes permet de dansguardian et détaillée Log Analyse des filtres et des rapports. Le Log l'analyse peut être exécuté sur une autre ordinateur et/ou peuvent produire des rapports en un seul lot mode (un exemple du fichier de script est fourni), ainsi que la gestion interactive dansguardian sur le serveur. Cette Webmin module fournit également des fichiers d'exemple pour la réalisation d'un Master→Slaves configuration où un seul site est administré que les autres sont simplement des "clones".

Contrairement à beaucoup d'autres Webmin modules, ce dansguardian Webmin module utilise beaucoup de JavaScript. En fait, il pourrait même ne pas être utilisable si Javascript est désactivé. Aussi, ce module nécessite beaucoup plus CPU et de RAM que les versions précédentes. Soyez prudent sur l'exécution sur le même ordinateur que dansguardian au cours des heures de production.
