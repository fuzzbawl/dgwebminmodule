Liste des groupes Filter
  

C'est la liste qui contient la cartographie des utilisateurs de filtrer les groupes. Il est utilisé par presque toutes les méthodes d'authentification plugin (il n'est pas auth plugin utilisé par des méthodes telles que IP-auth - que plutôt que d'identifier les ordinateurs des utilisateurs).

Il est utilisé seulement si il ya plus d'un filtre de groupe (c'est-à-dire filtergroups a une valeur supérieure à un). Dans les systèmes sans filtre de multiples groupes, tout le monde est toujours attribué à filtre groupe f1 Cette liste des fichiers n'est pas consulté.

Le format de chaque entrée est utilisateur=filter1-99 1-99, où sont le filtre groupes. (Habituellement, seule une poignée de groupes de filtre sont utilisés, mais dansguardian permet jusqu'à 99 groupes de filtre de manière à ne jamais être la cause d'une limitation.) À titre d'exemple, une entrée peut ressembler à:
danielg=filter2
