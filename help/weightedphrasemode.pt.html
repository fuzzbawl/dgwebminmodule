Weighted Phrase mode
  

Há 3 possíveis modos de operação:

  • 0 (desligado) = não use o recurso a todos os ponderada frase.
  • 1 (ligado, normal) = ponderada frase normal funcionamento, contar cada ponderada frase quantas vezes isso ocorre em uma página web.
  • 2 (sobre, singular) = suplente peso pontuação método de cálculo, contagem ponderada cada frase uma única vez [da primeira vez] que ocorre em uma página web.

Passando a frase ponderada recurso será completamente desligada lobotomize DansGuardian de "verdadeiro" (ou "conteúdo") filtragem web para pouco mais de fantasia (incluindo as expressões regulares) URL filtragem complementado por frases proibidas, bem como a possibilidade de modificação de conteúdo, o que não é muito diferente da web mais simples filtros.

DansGuardian suporta dois métodos diferentes de cálculo de peso, pontuação, o "normal" eo método do "singular" método. Cada método tem seus prós e contras. Muito boas páginas web pontuação será assim com qualquer método, e muito ruim páginas pontuação vai mal com qualquer método, mas middling páginas maio pontuação abaixo do seu limite com um método, mas acima do seu limiar com a outra. Por isso não mudar um sistema estabelecido DansGuardian para utilizar o outro método de ânimo leve.

Além disso, o peso escore dois métodos podem exigir um pouco diferente "naughtyness" limites, a fim de se comportar do mesmo modo. Se o valor que você escolheu para "naughtynesslimit" é apenas um palpite bruto, ela não importa. Mas se você sintonizado "naughtynesslimit" dentro de uma margem de apenas 5 ou 10, pode ser necessário re-sintonizar-la depois que você altere o peso escore método.
