Max contenido tamaño de la caché de exploración de archivos
  

Esto sólo se utiliza si se utiliza un escáner de contenido tales como plugin AV. Este es el tamaño máximo de archivo que se descarga la DG de modo que se pueden escanear o virus marcada. Este valor debe ser mayor o igual a maxcontentramcachescansize.

El tamaño es en Kilobytes - 2048 por ejemplo, = 2Mb
