afficher ou modifier l'échelle du système Plugin Configurations
  
  

Certaines de l'opération de dansguardian est sous-traitée de "plugins". Les plugins sont des bouts de code qui invoque dansguardian de fournir des fonctionnalités supplémentaires et sont presque complètement séparés dansguardian de la base. L'utilisation de plugins prévoit:

  • extensibility plus facile, même en théorie, permettre à l'utilisateur écrit des extensions
  • plus facile la maintenance du code source
  • une interface utilisateur plus compartimenté

Actuellement, les plugins sont utilisés pour fournir des fonctionnalités de:

  • authplugin (identifier les utilisateurs ou d'ordinateurs, d'affectation à un filtre de groupe)
  • downloadmanager (améliorer l'expérience de téléchargement de fichier)
  • contentscanner (filtrage supplémentaire, généralement anti-virus)
Chacun de ces plugins est une sous-configuration de son propre. Bien que la plupart de ces sous-configurations sont minimes et n'a jamais besoin de changer, de temps en temps l'un de ces sous-configurations de réglage n'est nécessaire. Utilisez cet outil lors d'un plugin sous-configuration besoins modification.
