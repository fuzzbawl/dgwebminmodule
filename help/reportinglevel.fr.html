Rapport d'accès interdit
  

Les options présentées ici pour refuser l'accès au Web de rapports n'ont pas d'incidence sur l'exploitation forestière.
Le niveau:

  • -1 = journal, mais ne pas bloquer (mode furtif)
  • 0 = Juste dire "Accès refusé"
  • 1 = rapport, mais pourquoi ne pas nier ce que la phrase
  • 2 = Rapport pleinement
  • 3 = Utilisez le modèle de fichier HTML (Access Denied Adresse ignorés)
