Ver ou Editar Filtro Grupo Configuração
  
  

Ver ou editar as definições da maior parte do per-filtro grupo opções de configuração. Cada filtro existente grupo exibe um "guia" de opções de configuração nesta interface de utilizador. Selecione a guia correspondente ao grupo que pretende ver ou alterar. (Todas as opções de configuração de cada dansguardianfN.conf filtro grupo config file estão em apenas uma guia, assim uma vez que você tenha encontrado o direito filtro grupo, você não vai precisar alternar guias novamente.)

O filtro do Grupo de nome é apresentado como a primeira opção no topo de cada aba. Na maioria dos casos, ela não deve ser alterado.

A definição actualmente activa é mostrado com uma borda em torno dela. Como você faz as alterações propostas, a fronteira pode mover.

Para cada opção, o "Conjunto"? checkbox mostra ou não alguma definição para a opção está presente no arquivo de configuração. (Isso permite que os casos de valores vazios como 0 ou "off" e da ausência completa de qualquer especificação para a opção devem ser distinguidas claramente.) Para mudar ou não uma opção configuração está presente, clique sobre a caixa de seleção.

Se uma opção configuração é (ou será) presente, o "Filtro Grupo Valor" área irá mostrar o seu valor. O "Filtro Grupo Valor" área perto do lado esquerdo terá uma borda em torno dele para que você possa mais facilmente discernir quais opções são (ou serão) presentes. Se, por outro lado a opção Definir? é não marcada, referência será feita, quer um sistema de escala o valor, ou para um valor padrão, se houver um. A fronteira irá deslocar-se para tanto o "Todos os grupos" área ou o "padrão" na área do direito.

Você não pode alterar qualquer destes valores com essa ferramenta, mas você pode ler o que eles estão. Para as opções na parte inferior de cada guia, usar o "Ver/Editar System-Wide Base Config" ferramenta e escolha o seu "Relatórios" na guia e definir valores globais do sistema, em seguida, voltar a essa ferramenta para fazer uso dos novos valores. Predefinições não pode ser mudado em nada.

(O "default" é o valor-compilados em valor que serão realmente utilizadas, se a opção não é especificado. É muitas vezes não o valor recomendado nem a um defeito que aparece nos ficheiros de configuração.)

Se a fronteira desaparece quando você desmarcar a Definir? caixa, isso significa que não existe nem um sistema de escala "todos os grupos" que nem um padrão, e, portanto, a opção será indefinido.

O «htmltemplate 'opção só será utilizada se o 'reportinglevel' opção está definida para "usar HTML arquivo modelo". Da mesma forma, o 'accessdeniedaddress' e 'nonstandarddelimiter' opção só será utilizada se o 'reportinglevel' opção é não definido como "uso HTML arquivo modelo". No entanto, estas relações não são reflectidas no visor. Em vez disso, todas as opções serão sempre exibidos embora alguns deles não podem ser utilizados.
