Configurar Config-&-Lista de Archivos Para varios grupos de filtro
  

Este proceso se debe realizar una vez al filtros múltiples grupos se aplicó por primera vez. Una vez que los filtros múltiples grupos se crean, este proceso no deben realizarse de nuevo.

La creación de todos los archivos de filtros múltiples grupos sin esta ayuda puede ser bastante intimidante. Esto hace que el proceso de instalación más simple.

Aunque es automatizado, el proceso es algo desestabilizador. Por lo tanto, no deben realizarse cuando se está ejecutando DansGuardian ni cuando debe usar DansGuardian para la producción en tan sólo unos minutos.

Se le pedirá para varios bits de información sabe lo que el proceso de cómo configurar filtros múltiples grupos de su entorno. Se le preguntó qué sheme general que desea utilizar, si desea dejar de lado un filtro para el grupo de "no" el acceso a la Web, si desea reservar un filtro de grupo para "todos" el acceso a la Web, lo que cada filtro de grupo debe ser nombre, y si varias listas deben almacenarse por filtro de grupo o filtro de todos los grupos deben utilizar una lista compartida.

Los posibles planes generales son:

  • 1: lista de archivos anidados

    esto significa cada una de las listas de archivos afectarán no sólo a su propio filtro de grupo, sino también todos los demás grupos de filtro a continuación (o superior) que. Así, por ejemplo si el filtro de los grupos "temps", "empleados", "administradores", y "ejecutivos", cualquier cambio de la excepción ... listas para "administradores" también automáticamente afectar "ejecutivos", y cualquier cambio en la prohibido ... listas para "administradores" también automáticamente afectan a los "empleados" y "tiempo".

  • 2: lista de archivos separados esto significa

    cada filtro grupo se dará propia completamente independiente de las listas, que no afecta ni se verán afectados por cualquier otro filtro de grupo

  • 3: lista de archivos comunes esto significa

    cada filtro grupo tiene sus propias listas, sin embargo, también una lista de referencias que es común filtro a todos los grupos. En otras palabras hay común "base" de listas en las que los grupos entonces elaborar filtro.
