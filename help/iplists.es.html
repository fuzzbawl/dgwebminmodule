Prohibido/deExcepción IP Listas
  

Estos archivos proporcionan una rápida y sucia forma para obtener DansGuardian especial para manejar las computadoras un modo particular, generalmente por períodos cortos de tiempo. Si encuentra un ordenador ha sido infectado por malware, estos archivos proporcionan una forma de instantánea apagar la computadora que el acceso a la Web hasta que se pueda limpiar.

(Estos archivos no son un substituto para la correcta configuración de "autoridades" métodos.)
