Number of Filter Groups

This is the keystone option that turns on the whole multiple filter groups mechanism in DansGuardian. Set it to the number of filter groups you want. Each of those filter groups will then be configured by a dansguardianfN.conf file where N is the filter group number (1-99).

If this option is set to only 1, the whole multiple filter groups mechanism will effectively be turned off. All user requests will always be assigned to filter group f1, and only dansguardianf1.conf will be meaningful.
