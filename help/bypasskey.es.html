Bypass utiliza la funcionalidad de un hash de clave secreta para garantizar la seguridad.

Con la presentación de informes de nivel 3 y de la plantilla HTML, Se proporciona la máxima seguridad mediante el uso de diferentes criptográfico de clave secreta cada vez. Este comportamiento de la elección de una nueva clave cada vez que se especifica en la omisión de esta opción (o como se especifique la cadena vacía).

Nivel de información con valores distintos de 3 y programas CGI que debe generar la URL de circunvalación, tales aleatorización no funcionará. Por lo tanto, es posible en lugar de generar una clave aleatoria cada vez para especificar explícitamente una clave que se utilizará cada vez. Debe ser más de 8 caracteres, por ejemplo
bypasskey = "Mary tenía un pequeño cordero." o bypasskey = '76b42abc1cd0fdcaf6e943dcbc93b826 '.

La misma funcionalidad se ofrece para las páginas que están infectados con el virus (y no negado por el contenido) por la opción de configuración infectedbypasskey, con la misma célula.
