Filtro lista de grupos
  

Esta é a lista que contém o mapeamento dos usuários para filtrar grupos. Ele é usado por quase todos os métodos auth plugin (é não plugin métodos utilizados pelas autoridades, como o IP-au - identificar computadores que, em vez de utilizadores).

É utilizado apenas se houver mais de um filtro grupo (isto é, filtergroups tem um valor maior do que um). Em sistemas sem filtros múltiplos grupos, toda a gente é sempre atribuído ao grupo filtro f1 e esta lista arquivo não é consultado.

O formato de cada entrada é usuário=filter1-99 1-99 onde estão o filtro de grupos. (Normalmente, apenas um punhado de grupos filtro são usados, mas DansGuardian filtro permite até 99 grupos de modo a não ser a causa de uma limitação.) Como um exemplo, uma entrada pode ser semelhante a este:
danielg=filter2
