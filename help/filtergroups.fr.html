Nombre de groupes de filtre
  

Cette option est la clé qui tourne dans l'ensemble filtres multiples groupes mécanisme dansguardian. Réglez-le sur le nombre de groupes que vous souhaitez filtrer. Chacun de ces groupes de filtre sera alors configuré par un dansguardianfN. conf file N est le numéro de groupe du filtre (1-99).

Si cette option est fixée à seulement 1, l'ensemble des filtres multiples groupes mécanisme seront effectivement mis hors tension. Toutes les demandes seront toujours affectés à filtre groupe f1, et seulement dansguardianf1.conf sera utile.
