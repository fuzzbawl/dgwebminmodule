Username identificação métodos
  
Você pode ter tantos métodos que você quiser e não apenas um. O primeiro deles será utilizado e, em seguida, se nenhum usuário foi encontrado, será utilizada a próxima.

Proxy Auth
Proxyauth é para quando proxy autenticação básica é usada (não bom para proxy transparente).

Ident Auth
Ident é para quando os outros não funcionam. Irá entrar em contato com o computador que a ligação veio e consulta-lo para o proprietário da ligação. Um ident servidor / daemon é necessário para ser executado no computador.
Consulte Identd para Win9x para um daemon identd para a plataforma Windows.
