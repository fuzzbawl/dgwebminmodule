anular o Maior Numbered Filtro Grupo de acesso irrestrito
  

Esta é uma opção para conveniência Webmin etc Embora ele é armazenado em dansguardian.conf, que actualmente significa nada para DansGuardian si.

É por vezes conveniente para ser capaz de atribuir qualquer utilizador a "livre" acesso à Web, a qualquer momento. Com múltiplas filtergroups, filtrar um grupo pode ser dado o pleno acesso à web. Então você pode atribuir qualquer usuário a este filtro grupo a qualquer momento.

Esta opção indica o filtro Grupos Instalação de reserva o seu mais alto numeradas filtro grupo para acesso ilimitado. Se é sobre / controlados, o maior grupo numerado filtro Será dado pleno acesso à web. Se, por outro lado, é desligado / desmarcado, o maior grupo numerado filtro grupo estará disponível para novas configurações.
